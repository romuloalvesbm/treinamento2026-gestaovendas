\# 📦 Projeto - Gestão Hexagonal



\## 📖 Sobre o Projeto

Este projeto é uma aplicação exemplo utilizando boas práticas de arquitetura e padrões modernos, com foco em manutenibilidade, escalabilidade e desacoplamento.



A solução foi construída com:



\- Arquitetura Hexagonal (Ports \& Adapters)

\- Entity Framework Core

\- Unit of Work

\- RabbitMQ (mensageria)

\- Serilog (logging estruturado)



\---



\## ⚙️ Tecnologias Utilizadas



\- .NET 10

\- Entity Framework Core

\- RabbitMQ

\- Serilog

\- SQL Server

\- Docker (opcional)



\---

\### ✔️ Arquitetura Hexagonal
